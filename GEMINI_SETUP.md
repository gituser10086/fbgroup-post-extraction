# FB Group 帖子采集器 - Gemini AI 集成指南

## 功能概述

扩展现在支持使用 **Google Gemini AI** 智能过滤帖子，可以自动识别：
- 💼 **有偿工作机会** - 识别付费工作
- 🎨 **自由职业/承包** - 识别自由职业和承包工作
- 🎓 **实习机会** - 识别实习职位
- ❤️ **志愿者/无偿** - 识别无偿和志愿者工作

## 快速开始

### 1. 获取 Gemini API 密钥

1. 访问 [Google AI Studio](https://aistudio.google.com/)
2. 点击 **"Get API key"** 按钮
3. 创建新的 API 密钥（免费层有配额限制）
4. 复制你的 API 密钥

### 2. 在扩展中配置

1. 打开 Facebook Group 页面
2. 点击扩展图标打开弹窗
3. **启用 AI 智能过滤** - 勾选复选框
4. **输入 Gemini API 密钥** - 粘贴你的 API 密钥
   - 密钥会本地保存，不会上传到任何服务器
   - 验证成功会显示 "✓ API 密钥有效"
5. **选择过滤类型** - 选择你想要的工作类型
6. **调整置信度阈值** (0.0-1.0)
   - 默认 0.7：保守过滤，准确率高
   - 0.5-0.6：平衡模式
   - 0.3-0.4：宽松模式，更多结果但可能有误

### 3. 开始采集

1. 在 Facebook Group 中采集帖子
2. 选择"采集帖子"或"追加"
3. 如果启用了 AI 过滤，系统会：
   - 先采集帖子
   - 然后用 AI 分析每条帖子
   - 只保留符合条件的结果

## 工作流程

```
采集帖子 → 本地关键词过滤 → AI 分类 → 显示结果
```

1. **采集** - 从 Facebook Group 抓取帖子内容
2. **本地过滤** - 按 filter-keywords.js 中的关键词过滤（如果配置）
3. **AI 分类** - Gemini 分析帖子内容，判断是否符合选定的工作类型
4. **显示结果** - 显示 AI 认可的帖子，并标注置信度百分比

## UI 说明

### 采集后的帖子显示

每条帖子会显示以下信息：
- **作者头像 + 名字** - 发布者信息
- **时间** - 发布时间
- **内容预览** - 帖子前 100 个字符
- **📊 统计** - 赞、评论、分享数量
- **🖼** - 图片数量（如果有）
- **🤖 置信度** - AI 对该帖子的置信度百分比（如果使用了 AI 过滤）

### 搜索/过滤

在采集后，可以在搜索框中输入作者名字或关键词进一步过滤结果。

## 高级配置

### filter-keywords.js

仍然支持本地关键词过滤：
- `FB_SCRAPER_FILTER_KEYWORDS` - 包含这些词的帖子会被排除
- `FB_SCRAPER_REQUIRED_KEYWORDS` - 帖子必须包含至少一个这些词

AI 过滤在本地关键词过滤之后执行。

### 导出结果

支持以下格式导出：
- **JSON** - 完整数据，包含 AI 分类信息
- **CSV** - 电子表格格式
- **TXT** - 易读的文本格式

## API 配额和限制

Google Gemini API 有免费配额：
- **免费层** - 每分钟最多 60 个请求
- **付费层** - 按使用量计费

扩展采用了优化策略：
- 批量处理，防止超过速率限制
- 批次间延迟，避免 API 限流
- 本地缓存采集结果，重复利用

## 常见问题

### API 密钥无效怎么办？
- 检查密钥是否正确复制（没有多余空格）
- 确认密钥在 Google AI Studio 中仍然有效
- 检查 Google 账户是否启用了 Gemini API

### AI 过滤不准确？
- 调整置信度阈值（降低阈值会有更多结果）
- 确保帖子内容足够完整（不要设置太低的采集字符数）
- 使用 filter-keywords.js 配合本地关键词过滤

### 为什么采集很慢？
- AI 分析需要网络请求，会比纯本地过滤慢
- 建议先用本地关键词过滤减少帖子数量
- 可以增加采集超时时间

### 如何禁用 AI 过滤？
- 取消勾选"启用 AI 智能过滤"复选框
- 或者删除 API 密钥

## 隐私和安全

- **API 密钥本地存储** - 不会发送到任何第三方服务器
- **帖子内容** - 仅发送给 Google Gemini API（你的 API 密钥）
- **Google 隐私政策** - 适用于 API 请求
